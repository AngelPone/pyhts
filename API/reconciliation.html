reconciliation¶

pyhts.reconciliation.wls(hierarchy, error=None, method='ols', weighting=None, constraint_level=- 1)¶

function for forecast reconciliation.

Parameters

  • hierarchy (Hierarchy) – history time series.

  • error (Optional[ndarray]) – insample error.

  • method (str) – method used for forecast reconciliation, e.g. ols, wls, mint.

  • weighting (Union[str, ndarray, None]) – method for calculating weight matrix used in reconciliation method, e.g. covariance matrix in mint or wls.

  • constraint – If some levels are constrained to be unchangeable when reconciling base forecasts.

  • constraint_level (int) – Which level is constrained to be unchangeable when reconciling base forecasts.

Return type

ndarray

Returns

reconciled forecasts

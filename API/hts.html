Hts¶

class pyhts.hts.Hts¶

Class for hierarchical time series, can be constructed from cross-sectional multivariate time series like hierarchical or grouped time series and temporal hierarchies 1.

forecast(h, base_method='arima', hf_method='comb', weights_method='ols', weights=None, variance='shrink', parallel=False, constraint=False, constrain_level=0)¶
Parameters

  • h – forecast horzion

  • base_method – method for generate base forecast

  • hf_method – method for hierarchical forecast: comb, bu, td, mo

  • weights_method – ols, wls, mint

  • weights – nseries, custom_matrix

  • variance – cov, var, shrink

  • parallel – Bool

  • constraint – Bool, if constrained

  • constrain_level – int, if constrained, which level

Returns

Hts: reconciled forecast

classmethod from_hts(bts, m, characters=None, nodes=None)¶

Construct hts from cross-sectional hierarchical time series.

Parameters

  • bts (Union[ndarray, DataFrame]) – \(m\times T\) bottom level series.

  • m (int) – frequency of time series

  • characters (Optional[List[int]]) – length of characters of each level, for example, “ABC” ‘A’ represents ‘A’ series of first level, ‘B’ represents ‘B’ series under node ‘A’, ‘C’ represents ‘C’ series under node ‘AB’. So the value of parameter is [1,1,1]

  • nodes (Optional[List[List]]) – List of list to demonstrate the hierarchy, see details.

Return type

Hts

Returns

Hts object.

  • pred – forecasts.

  • hist – history observations

  • s – seasonality

  • levels (Union[int, None, List]) – which level.

  • measure (Optional[List[str]]) – mase, mse is supported for now. e.g. [‘mase’], [‘mse’, ‘mase’]. if None, mase is calculated.

Returns

accuracy_base(real, pred, hist=None, levels=None, measure=None)¶

calculate forecast accuracy of base forecast

Parameters

  • real – real future observations, array-like of shape (forecast_horizon, m)

  • pred – forecast values, array-like of shape (h, n)

  • levels (Union[int, None, List]) – which level.

  • hist – history time series.

  • s – seasonality of series

  • measure (Optional[List[str]]) – mase, mse is supported for now. e.g. [‘mase’], [‘mse’, ‘mase’]. if None, mase is calculated.

Return type

DataFrame

Returns

forecast accuracy of base forecasts.
